# nocemcu-restful-server

flash your esp8266 module with the nodemcu firmware and upload the files:

 * form.html
 * gpio.html
 * gpio.json
 * gpio.lua
 * index.html
 * init.lua
 * luadown.png
 * README.md
 * restful-server.lua
 * rest.lua
 * style.css

compile the restful-server.lua and run it.

open http://ip-or-name-of-esp/
